## Summary

Нужно внести три изменения на главной странице сайта ОДИ:

- временно скрыть раздел `Сейчас в продаже` и пункт `Продажа` в навигации, сохранив код раздела для будущего возврата.

## Context

- Галерея построенных объектов собирается в `apps/web/src/data/gallery.ts`.
- В локальном коде уже есть функция `movePriorityGalleryItems`, которая похожа на нужную перестановку. При выполнении нужно сначала проверить фактический порядок и не дублировать лишнюю логику.
- Пункт `Продажа` сейчас задан в `navLinks` внутри `apps/web/src/components/SiteHeader.tsx`.
- Раздел `Сейчас в продаже` отрисовывается в `apps/web/src/pages/Home.tsx` через `GallerySection` и данные `saleHouseItems`.
- Попап обратного звонка открывается из `SiteHeader` и использует общий `Modal` + `LeadForm`.
- Стили модалок находятся в `apps/web/src/styles/components.css`, стили формы попапа — в `apps/web/src/styles/sections.css`.

## Assumptions

- Под `п. Холмогоровка 84,2 м²` имеется в виду объект с id `Одноэтажный дом 84,2 м²`.
- Под `п. Малое Васильково 84,2 м²` имеется в виду объект с id `Одноэтажный дом 84,2 м² (Малое Васильково)`.
- Автоподстановка `+7` нужна именно в попапе `Заказать звонок`; остальные формы не меняются без отдельного решения.
- Раздел продажи нужно скрыть визуально и из навигации, но не удалять данные, компонент галереи и тексты, чтобы позже вернуть раздел одним небольшим изменением.
- Backend, API, Telegram-доставка, `.env.example`, deploy и зависимости не трогать.
