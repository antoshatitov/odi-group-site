# Памятка по доступу к VPS Timeweb

## Сервер

- IP: `109.73.195.213`
- SSH-доступ для `root`: отключен

## Пользователи

- `anton` — основной административный доступ, полный `sudo`
- `codex` — ограниченный доступ без `sudo`

## Быстрый вход

Для входа используются локальные SSH-алиасы:

```bash
ssh timeweb-anton
ssh timeweb-codex
```

Детали локальной SSH-конфигурации и пути к ключам хранятся вне репозитория.

## Проверка доступа

Проверить административный доступ:

```bash
ssh timeweb-anton
sudo -n true
```

Проверить ограниченный доступ:

```bash
ssh timeweb-codex
sudo -n true
```

Для `codex` команда `sudo -n true` должна завершаться отказом.

## Полезные замечания

- Вход по паролю по SSH отключен, доступ работает только по ключам.
- Вход под `root` по SSH отключен.
- Для административных задач используйте `anton`.
- Для обычной работы Codex на сервере используйте `codex`.

## Аварийный доступ

Если SSH по какой-то причине недоступен, используйте консоль/панель провайдера Timeweb Cloud.
Аварийные учетные данные хранятся вне репозитория.
